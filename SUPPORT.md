# Support

If you need help using **Jeninnet.FileQuery**, please follow these steps:

1. Read the documentation located in the `/docs` directory.
2. Review the examples in the `/samples` folder.
3. Open a GitHub Discussion if you have questions or need guidance.

For bugs or unexpected behavior:

- Please open a GitHub Issue and include as much detail as possible (steps to reproduce, expected behavior, environment info, etc.).
